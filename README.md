Workflow Overview
- Fork repository gốc về 
- Clone code từ fork của bạn
- Tạo nhánh họ-tên
- Hoàn thành bài tập
- Push lên nhánh họ-tên
- Tạo pull request

Detailed Steps
1. Fork và Clone
```
git clone https://github.com/repository của mình/front-end
cd front-end
```
2. Tạo nhánh
```
git checkout -b firstname-lastname
```
3. Commit và Push
```
git add .
git commit -m "Your commit message"
git push origin firstname-lastname
```
4. Tạo Pull Request
- Tới repo https://github.com/huy14032003/front-end
- Click "Pull Requests" → "New Pull Request"
- Click "compare across forks"
- Lựa chọn nhánh
- Create pull request
Done oke
